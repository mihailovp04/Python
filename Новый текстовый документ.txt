Проект "Конвертер валют"
Описание проекта
Разработать простое консольное приложение для конвертации валют. Пользователь вводит сумму в одной валюте, выбирает валюту, в которую хочет перевести деньги, и программа выводит результат.

Использование тем из курса
Темы 1-3 (Введение в Python, Типы данных, Коллекции):
Основы языка, использование строк, чисел и словарей для хранения курсов валют и ввода/вывода данных.
Тема 4-5 (Методы и функции, Создание собственных функций):
Разработка функций для конвертации валют и получения курсов из словаря.
Тема 6 (Условные инструкции):
Управление потоком выполнения программы в зависимости от выбора пользователя.
Тема 7 (Циклы):
Циклы для повторения операции конвертации по желанию пользователя.
Тема 9 (Взаимодействие с файлами данных):
Опционально, сохранение и загрузка курсов валют из файла для избежания их ручного ввода каждый раз.
Темы 10-11 (Классы, объекты и принципы ООП):
Создание класса для конвертера валют, что позволит упорядочить код и упростить расширение функционала в будущем.

Проект "Планировщик задач"
Описание проекта
Создать простое консольное приложение для управления задачами. Пользователи смогут добавлять, удалять, и просматривать задачи на определенные даты, что поможет организовать их ежедневные дела.

Использование тем из курса
Темы 1-3 (Введение в Python, Типы данных, Коллекции):
Использование базовых типов данных и коллекций для хранения информации о задачах.
Тема 4-5 (Методы и функции, Создание собственных функций):
Создание функций для добавления, удаления и отображения задач.
Тема 6 (Условные инструкции):
Применение условных инструкций для управления выбором пользователя в меню приложения.
Тема 7 (Циклы):
Использование циклов для обработки ввода пользователя и повторения действий меню.
Тема 9 (Взаимодействие с файлами данных):
Сохранение и загрузка списка задач из файла для постоянного хранения данных.
Темы 10-11 (Классы и объекты, ООП):
Опционально, структурирование кода с использованием классов для представления задач и менеджера задач.
Этот проект позволит вам практиковать и закрепить навыки программирования на Python, включая работу с файлами, базовое ООП и интерфейс командной строки.
